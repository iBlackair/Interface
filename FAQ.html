FAQ

  • Q: Why font is so big?

    • A: You have to install dooztr_0.TTF from system folder. Right click on it and hit "Install".

  • Q: How to move panels?

    • A: Drag it by clicking on left side of the panel. There is transparent texture.

  • Q: How to bind keys on 4+ panel?

    • A: Find in system folder file "Option.ini", open it and add this.

    [HiddenShortcuts]

    4_1=Q

    4_1_mod1=ALT

    Meaning: 4 - number of panel, 1 - number of shortcut, mod1 - ALT, CTRL or SHIFT
